## PARAM-1: 한국 시장 (KR)

### KR-1. 중개업 생태계 구조

| 항목 | 수치/특성 | 전략 함의 |
|------|-----------|-----------|
| 전국 공인중개사 수 | 약 **55만 명** 자격증 보유, 활성 사무소 ~10만 개 | 시장 크지만 파편화 극심 → 길드 컨셉 희소성 작동 |
| 1인 사무소 비율 | **80% 이상** 혼자 운영 | 도구를 대신 써줘야 함. 셀프서비스 안 됨 |
| 평균 연령 | 50대 이상 다수 | 모바일 우선이지만 복잡한 UI 거부 |
| ChatGPT 인지율 | 높지만 **실사용율 낮음** | "아는데 못 쓰는" 갭 → 대리 실행이 상품 |
| 폐업률 | 연간 20~30% | 살아남은 중개인은 마케팅 필요성 절감 |

### KR-2. 한국 부동산 디지털 마케팅 실상

```
중개인이 실제로 쓰는 채널 (중요도 순):
1위. 네이버 부동산 광고 (매물 등록 유료화)
2위. 네이버 블로그 (SEO 트래픽)
3위. 카카오톡 채널 (단골 관리)
4위. 유튜브 쇼츠 (요즘 급증)
5위. 당근마켓 (지역 매물 직거래)
```

**핵심 인사이트**: 네이버 블로그가 여전히 부동산 정보 검색의 1차 경로. "아파트명 + 매매" 검색 시 블로그 노출이 직방/다방보다 상위. → **워키토키→블로그 자동화가 핵심 무기**

### KR-3. 카카오/네이버 생태계 의존도

| 툴 | 중개인 활용 방식 | namoneygoal 연동 포인트 |
|----|----------------|------------------------|
| **카카오톡 채널** | 단골 고객 연락처 관리, 공지 발송 | PR 버튼 → 카카오 채널 연결 (이미 구현) |
| **카카오 알림톡** | 계약일 리마인더, 입금 확인 | 정산 자동 알림, 온보딩 안내 |
| **네이버 블로그** | 매물 홍보, 지역 정보 | 워키토키 → 블로그 자동 발행 |
| **네이버 스마트플레이스** | 사무소 위치/리뷰 | 슬롯 명함 → 스마트플레이스 연동 |
| **네이버 MY플레이스** | 방문자 리뷰 | 브랜드 신뢰도 누적 |

### KR-4. 한국형 프랜차이즈 부동산 사례 vs namoneygoal

| 기존 사례 | 모델 | 문제점 |
|-----------|------|--------|
| ERA코리아, 센추리21코리아 | 글로벌 프랜차이즈 직수입 | 가맹비 수천만 원, 간판만 주는 구조 |
| 직방PRO | SaaS 구독 | 스스로 써야 함, 학습 부담 |
| 부동산114 | 데이터 플랫폼 | 마케팅 지원 없음 |
| **namoneygoal** | **길드 멤버십 + 대행** | 12슬롯 한정, 본사가 직접 운영해줌 |

**포지셔닝 갭**: "도구를 파는" SaaS가 아니라 **"운영을 대신해주는" 운영대행** 모델. 한국 시장에서 이 포지션은 비어있음.

### KR-5. 가격 민감도 파라미터

```
한국 소상공인 SaaS 수용 가격대:
- 월 3만 원 이하: 거의 무조건 씀 (ChatGPT 수준)
- 월 3~10만 원: 효과 증명 필요
- 월 10만 원 이상: "가성비" 또는 "허세" 둘 중 하나로 팔아야 함

namoneygoal 월 10만 원 = "ChatGPT(3만) + 나머지(7만)"로 프레이밍
→ "ChatGPT는 어차피 쓰는데, 거기에 7만 원만 더 내면 사무실이 생긴다"
```

### KR-6. 규제 파라미터

| 규제 | 내용 | 설계 영향 |
|------|------|-----------|
| 공인중개사법 | 중개업무 광고 시 등록번호 표시 의무 | 명함/블로그 템플릿에 등록번호 필드 필수 |
| 부동산거래신고법 | 실거래가 공개 의무 | 공공 API 연동으로 시세 자동 표시 |
| 개인정보보호법 | 고객 연락처 수집/보관 | CRM 데이터 암호화, 동의 절차 |
| 광고규제 | "최고/최저" 등 과장 표현 금지 | 템플릿에서 해당 표현 사전 필터링 |

## PARAM-2: 시흥 (SH)

### SH-1. 시장 규모 파라미터

```
인구:     587,106명 (2025.7 기준) → 60만 돌파 임박
외국인:   71,757명 (12.2%) — 전국 평균 3배. 다문화 니즈
세대수:   약 23만 세대 (추정)
중개사무소: 추정 800~1,200개소
경쟁밀도: 사무소당 담당 세대 약 200~280세대
```

**12슬롯 한정의 근거**: 1,000개 사무소 중 12개 = 1.2% 점유. 희소성 유지하면서도 충분한 리크루팅 풀.

### SH-2. 권역별 슬롯 전략

```
시흥 5대 권역 = 5가지 캐릭터

┌──────────────────────────────────────────────────────────┐
│  권역         │  단지 특성          │  슬롯 페르소나       │
├──────────────────────────────────────────────────────────┤
│  배곧신도시   │  해안+서울대+고가   │  "프리미엄 전문가"   │
│  정왕동       │  공단배후+노후+외국인│  "실속파 로컬전문가" │
│  은계지구     │  생태전원+중산층    │  "자연친화 라이프"   │
│  장현지구     │  신시가지+트리플역세│  "미래가치 선도자"   │
│  월곶/소래    │  해안+개발예정+복합 │  "뉴웨이브 개척자"   │
└──────────────────────────────────────────────────────────┘
```

**슬롯 배분 전략**:
- 권역당 최대 2~3슬롯 (초과경쟁 방지)
- slot01 (김남원) = 장현/은계 앵커
- slot02 (권영숙) = 장현지구 유승한내들 전담
- 다음 모집: 배곧 1, 정왕 2, 월곶 1 우선

### SH-3. 시흥 특유 개발 호재 타임라인

```
2026 ──────────────────────────────────────────────── 2030+

2026.3  서해선 완전개통 (서화성~원시)
2026    정왕권 노후아파트 준공 30년차 → 정비사업 수요 폭발
2027    장현지구 입주 본격화
2028~   신안산선 개통 (지연 중, 광명역~여의도)
2028~   월곶~판교 복선전철 개통 예상
2030+   GTX-C 연장(시흥/안산) 불투명
```

**전략 타이밍**: "기다리는 기간(2026~2028)" 동안 디지털 브랜딩 확보 → 개통 후 폭발적 수요 선점.

→ **슬롯 모집 카피**: "신안산선 개통 전에 이미 유명해져 있어야 한다"

### SH-4. 시흥 디지털 갭 파라미터

```
현황:
- 시흥공인중개사협회 지회에서 이미 AI 마케팅 교육 실시 중
- 교육 내용: 블로그 자동작성, 유튜브 쇼츠, 챗봇 상담
- 결과: "교육은 받았는데 혼자는 못 하겠다"는 반응 다수

→ 정확히 namoneygoal의 페인포인트:
   교육(수요) O + 실행력(공급) X = 대행 시장
```

### SH-5. 시흥 커뮤니티 파라미터

| 채널 | 특성 | 연동 전략 |
|------|------|----------|
| **시맘수 (시흥맘 카페)** | 지역 최대 커뮤니티, 집 정보 질문 多 | 슬롯 중개인이 카페 전문가로 활동 → 유입 |
| **당근마켓 시흥** | 매물 직거래 활성 | 슬롯 명함 링크 당근 프로필 연동 |
| **네이버 지역 검색** | "시흥 장현동 아파트 매매" 검색 | 블로그 SEO 타겟 키워드 권역별 세팅 |
| **카카오톡 지역 단톡방** | 아파트 단지별 입주민 단톡 | 워키토키로 단톡용 콘텐츠 즉시 생성 |

### SH-6. 시흥 경쟁 포지셔닝

```
광명 (+6.46%): 비싸고 이미 포화
안산 (+3.23%): 성장성 낮음
부천 (+5.16%): 서울 수요 흡수, 별도 생태계
시흥 (+4.71%): 개발 호재 多, 디지털 마케팅 선점 기회 ← 우리
```

## 두 파라미터가 설계를 어떻게 바꾸는가

### 기존 글로벌 벤치마크 → KR × SH 필터 적용

| 글로벌 표준 | KR 필터 적용 | SH 필터 추가 적용 |
|-------------|-------------|------------------|
| CRM (리드 관리) | 카카오톡 채널 기반 리드 관리 | 단지별/권역별 리드 분류 (배곧/정왕/은계...) |
| 콘텐츠 자동화 | 네이버 블로그 SEO 자동 발행 | 시흥 권역 키워드 자동 삽입 ("장현동 아파트", "배곧 매매") |
| 온보딩 위자드 | 카카오 로그인 (주민증번호 X) | 중개사등록번호 입력 → 공공데이터 자동 검증 |
| 정산/결제 | Toss Payments 정기결제 (가상계좌 포함) | 현금영수증 발행 (소상공인 세금 민감) |
| 교육/LMS | 카카오 채널 알림 기반 마이크로러닝 | 신안산선/월판선 호재 업데이트 콘텐츠 |
| 분석 대시보드 | 네이버 블로그 조회수 연동 | 시흥 실거래가 API 연동 → 시세 자동 표시 |
| 대행 서비스 (없음) | **핵심 차별화: 본사가 직접 운영해줌** | 권역 전담 → "장현동 전문 AI 중개인" 포지셔닝 |

## 이 파라미터가 결정하는 개발 우선순위 재조정

| 기존 우선순위 | KR×SH 보정 후 | 이유 |
|---------------|---------------|------|
| Phase 1: 온보딩 | **Phase 1: 네이버 블로그 자동발행** | 중개인이 즉시 체감하는 가치 = 블로그 |
| Phase 2: 정산 | Phase 2: 온보딩 + 정산 (병행) | 결제는 이미 카카오페이 익숙 |
| Phase 3: CRM | **Phase 3: 권역별 키워드 SEO 엔진** | 시흥 특화 SEO가 CRM보다 선행 가치 |
| Phase 4: 대시보드 | Phase 4: CRM + 당근/카카오 연동 | 리드는 당근/카카오에서 옴 |
| Phase 5: 확장 | Phase 5: 실거래가 API + 호재 알림 | 시흥 전문성 = 공공데이터 활용 |
