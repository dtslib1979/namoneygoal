## 1. 4대 헌법 변경 (v1 → v2)

### § 1. 길드원·길드장 모두 **LLM 만 사용**

| 구분 | v1 | v2 |
|---|---|---|
| 슬롯원 (1~12) | "AI 쓰는 중개인" 추상 | **LLM 채팅만 허용. 코드 접근 금지** |
| 김남원 (앵커) | 지역 영향력자 | **앵커도 동일 — LLM only. 모든 수정은 PR 버튼** |
| 박씨 (HQ) | 운영자 | 운영자 + Tier B 멘토 |

**보호 가드:**
- 슬롯원이 자기 사이트 직접 못 만짐
- 모든 수정 → PR 버튼 → 박씨/Buckley 처리
- staff/walkie 도 음성+텍스트 만, 코드 안 만짐
- 이 원칙이 깨지면 = "허세 소비재" 가치 붕괴

---

### § 2. 검색·시세 엔진 **ChatGPT → Perplexity**

| 항목 | ChatGPT | Perplexity |
|---|:--:|:--:|
| 실시간 시세/분양 검색 | ❌ cutoff | ✅ 실시간 웹 |
| **출처 인용** (책임 회피) | ❌ | ✅ 자동 citation |
| 네이버 블로그/카페 인덱싱 | ⚠️ 약함 | ✅ 한국어 출처 강함 |
| 가격 | $20/월 | $20/월 |

**결정타**: 중개사가 "이 매물 시세 얼마요?" 답할 때 **출처 없이 GPT가 만든 숫자** = 사고 위험. Perplexity는 출처 자동 첨부 → 책임 회피.

**가격 변경:**
```
v1:  ChatGPT 3만 + 남원골 7만 = 월 10만
v2:  Perplexity Pro 3만 + 남원골 7만 = 월 10만 (총액 동일)
```

---

### § 3. 정체성 **"AI 쓰는 중개인" → "네이버 생태계 최고 전문가"**

**이유:**
- 시흥 부동산 = **네이버 블로그 + 카페 + 부동산 + 지도** 가 모든 채널
- ChatGPT 쓰는 중개인은 흔함
- **네이버 SEO + 시세 검색 자동화** 하는 중개인 = 희귀
- 글로벌 벤치마크에 없는 차별 포지션

**카피 변경:**
| 위치 | v1 | v2 |
|---|---|---|
| `<title>` | AI 쓰는 중개인 길드 | **네이버 생태계 전문가 길드** |
| 히어로 | "말하면 블로그가 된다" | **"말하면 네이버에 박힌다"** |
| concept | AI 리터러시 허세 | 네이버 생태계 + LLM 도구 마스터 |

---

### § 4. **2-Tier 트랙** (LLM 기본 / Agent 옵션)

```
┌─────────────────────────────────────────────────────────┐
│  Tier A · LLM TRACK (기본 · 12 슬롯 전원)               │

│  · PR 버튼 (수정 위탁 → 박씨/Buckley)                    │

┌─────────────────────────────────────────────────────────┐
│  Tier B · AGENT TRACK (선택 · 1~2명)                    │
│  ──────────────────────────────────────────────         │
```

## 2. 네이버 생태계 도구함 (신규 섹션)

```
시흥 부동산 = 네이버 4채널이 전부
─────────────────────────────────
  네이버 블로그   →  AI 워키토키 자동 발행 + Perplexity 시세 인용
  네이버 카페     →  지역 부동산 카페 모니터링 (Perplexity 검색)

  네이버 지도     →  슬롯 사무실 + 매물 위치 임베드
```
